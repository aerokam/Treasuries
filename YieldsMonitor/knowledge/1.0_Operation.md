# 1.0 Yields Monitor Overview

## Purpose
The Yields Monitor provides real-time and historical visualization of Treasury (Nominal) and TIPS (Real) yields. It is designed for intraday precision and long-term trend analysis.

## Data Architecture

**2D, 10D ranges**: Fetch intraday data from CNBC GraphQL API (`1D` and `5D` timeRange).

**1Y, 2Y, 3Y, 10Y, ALL ranges**: 
- Fetch historical baseline from R2 (`Treasuries/yield-history/{symbol}_history.json`) — daily closes updated by snapHistory.js
- Append latest intraday yields from CNBC `5D` feed to ensure "Latest Yield" and "Day Change" reflect current market data

**If R2 fetch fails**: Returns null; app displays "No data available." To fix: ensure R2 CORS headers are set to allow browser requests, or implement a backend proxy.

**No local fallback code**: App fetches only from remote sources (CNBC, R2).

## Tab Architecture
The monitor is organized into three analysis views:
- **Time Series**: Individual maturity charts with intraday shading and historical trending. (See 2.1)
- **Yield Curves**: Snapshot comparison of curves at the start vs. end of the range. (See 2.2)
- **Breakeven Inflation**: Visualization of market-implied inflation expectations. (See 2.3)

## Yield Change Calculation

The "day change" shown next to each symbol (e.g. `−0.016%`) is calculated as:

```
change = latest_yield − closeP.yield
```

**`closeP` selection** — walks backward through `calculationData` (prefers `_5D` cache over `_1D`) to find the last bar from the previous trading day where the bar timestamp is at **exactly 17:00 ET** or earlier (condition: `hour < 17 || (hour === 17 && minute === 0)`). This anchors the reference to the official end-of-trading-day close.

The tooltip shows: `Since MM/DD/YYYY, 17:00 ET close (X.XXX%)`.

### Deviation from CNBC

CNBC's displayed yield change uses a different reference: the `previous_day_closing` field from their Tradeweb quote service. This value reflects an **earlier settlement quote** — apparently a mid-afternoon official price (~14:00–15:51 ET range on observed days) — rather than the 5 PM trading close.

**Example (2026-04-08, US1Y):**
| Source | Reference time | Reference yield | Change vs. 3.677% |
|---|---|---|---|
| This app | 17:00 ET | 3.693% | −0.016% |
| CNBC | ~15:00 ET (Tradeweb settlement) | 3.685% | −0.008% |

The discrepancy (~0.008%) is expected and intentional. Our 17:00 reference is the actual trading close; CNBC's is an earlier official settlement publication (likely Tradeweb's intraday closing quote, distinct from end-of-day trading).

## Shared Interaction & Navigation
- **Timezone**: All displays are in **Eastern Time (ET)**.
- **Precision**: Yields are displayed with 3 decimal places (e.g., `4.321%`).
- **Panning & Zooming**: 
    - Mouse Wheel: XY zoom.
    - Ctrl+Scroll: X-axis zoom only.
    - Shift+Scroll: Y-axis zoom only.
    - Keyboard (Arrows, +/-): Panning and zooming around view center.
- **Y-Axis Auto-Rescale**: Automatic rescaling to visible data bounds ensures maximum vertical resolution during navigation.
- **Sync Zoom & Pan**: When enabled (default), zooming or panning any time series chart automatically synchronizes the X-axis (time) of all other active charts. This allows for easy comparison of yield movements across different maturities. Y-axis scaling remains independent per chart to maintain optimal resolution for each symbol's yield range.
