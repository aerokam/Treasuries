# 2.2 Yield Curves Specification

## Purpose
Visualizes the shift and twist in the yield curves over the active time range.

## Snapshots
Two datasets are plotted for each curve (TIPS and Nominal):
- **Start of Range**: Dashed blue line — the first data point in the active time range (or custom start date window).
- **Latest / End**: Solid red line — the last data point in the active time range (or custom end date window).

Both snapshots derive directly from `rangeData[sym]`, so they automatically reflect whichever range or custom dates are active.

## Labeling
Datasets are labeled with the precise ET timestamp of the actual data point they represent (e.g., `04/08 00:00 ET` for a daily close, `04/08 09:30 ET` for intraday).

## Custom Date Range
When the **Custom** range is active (see 2.1), the Start snapshot reflects the first available data point on or after the specified start date, and the End snapshot reflects the last available point on or before the specified end date. No additional code changes are needed in the yield curve logic — the filtering is done upstream in `fetchOne()`.
