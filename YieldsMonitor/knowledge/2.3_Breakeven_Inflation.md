# 2.3 Breakeven Inflation (BEI) Specification

## Purpose
Visualizes the Breakeven Inflation (BEI) curve, representing the market's expected inflation over various time horizons. This is the spread between Nominal and TIPS yields of equivalent maturity.

## Calculation Logic
- **Formula**: `BEI (%) = Nominal Treasury Yield (%) - TIPS Real Yield (%)`
- **Maturity Pairs**: The curve is plotted for the following five key maturities where both Nominal and TIPS yields are available:
    - **1-Year**: `US1Y` (Nominal) minus `US1YTIPS` (TIPS)
    - **2-Year**: `US2Y` (Nominal) minus `US2YTIPS` (TIPS)
    - **5-Year**: `US5Y` (Nominal) minus `US5YTIPS` (TIPS)
    - **10-Year**: `US10Y` (Nominal) minus `US10YTIPS` (TIPS)
    - **30-Year**: `US30Y` (Nominal) minus `US30YTIPS` (TIPS)

## Curve Snapshot Logic
Consistent with the Yield Curves tab (2.2), the BEI tab displays two snapshot curves based on the active time range (**2D**, **10D**, **1Y**, **2Y**, **3Y**, **10Y**, **ALL**, or **Custom**):
1.  **Start of Range** (Dashed Blue Line): Calculated using the first available data point for each pair within the selected range (or custom start date).
2.  **Latest / End** (Solid Red Line): Calculated using the last available data point for each pair within the selected range (or custom end date).

When the **Custom** range is active, filtering is applied upstream in `fetchOne()` so no additional BEI logic is required.

## Chart & UI Features
- **X-Axis**: Term to Maturity (1Y, 2Y, 5Y, 10Y, 30Y).
- **Y-Axis**: Breakeven Inflation Rate in percentage (%).
- **Interaction**:
    - Supports XY zoom and pan (Mouse/Keyboard/Sidebar).
    - **Tooltips**: Unified tooltips show the snapshot time (ET) and the calculated BEI for each maturity on hover.
- **Responsiveness**: The chart is updated automatically when active symbols, the time range, or the active tab changes.
