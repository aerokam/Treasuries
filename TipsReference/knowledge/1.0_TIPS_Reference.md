# 1.0 TIPS Reference — App Specification

## Purpose

A standalone read-only reference table for all live TIPS issues. Combines the data from the **Ref** and **Mkt** popups in TIPS Ladder Manager into a single unified view, adding computed index ratio and adjusted price columns.

---

## Data Sources

Same R2 bucket used by TIPS Ladder Manager:

| File | URL | Role |
|------|-----|------|
| `YieldsFromFedInvestPrices.csv` | `BASE_URL/YieldsFromFedInvestPrices.csv` | Settlement date (row 1), CUSIP / maturity / coupon / datedDateCpi / price / yield |
| `RefCPI.csv` | `TIPS_URL/RefCPI.csv` | date → refCpi lookup series |
| `TipsRef.csv` | `BASE_URL/TipsRef.csv` | CUSIP → datedDate, baseCpi (Dated Date Ref CPI) |

Constants:
```
R2_ROOT  = https://pub-ba11062b177640459f72e0a88d0261ae.r2.dev
BASE_URL = R2_ROOT/Treasuries
TIPS_URL = R2_ROOT/TIPS
```

## Columns (display order, left → right)

| # | Column | Source | Alignment | Format |
|---|--------|---------|-----------|--------|
| 1 | CUSIP | yields CSV | left | string |
| 2 | Maturity | yields CSV | left | MM/DD/YYYY |
| 3 | Coupon | yields CSV | right | 3 decimal places, % (e.g., 0.125%) |
| 4 | Dated Date | TipsRef.csv | left | string (YYYY-MM-DD) |
| 5 | Dated Date Ref CPI | TipsRef.baseCpi | right | 5 decimal places |
| 6 | Price | yields CSV | right | 3 decimal places |
| 7 | Index Ratio | computed | right | 5 decimal places |
| 8 | Adj Price | computed | right | 3 decimal places |
| 9 | Yield | yields CSV | right | 3 decimal places, % |

### Computed Fields

```
indexRatio = settlementRefCPI / baseCpi
adjPrice   = price × indexRatio
```

Where `settlementRefCPI` = `lookupRefCpi(refCpiRows, settlementDate)`.

`lookupRefCpi` returns the most recent refCpi entry on or before the given date. This matches the Treasury daily interpolation method referenced in TIPS Basics.

## Header

Fixed header bar contains (left → right):
- **← Portal** link (standard portal-link class)
- **Title**: "TIPS Reference"
- **Settlement Date** meta item (from row 1 of yields CSV, formatted MM/DD/YYYY)
- **Settlement Ref CPI** meta item (5 decimal places)
- **Status** message (loading / error, pushed to right via margin-left: auto)
- **Export CSV** button (disabled until data loads)

---

## Table Behavior

### Sorting
- Click a column header to sort ascending; click again to toggle descending.
- Active sort column shows ↑ or ↓ indicator.
- Default sort: Maturity ascending.
- Sort types: text columns (CUSIP, Maturity, Dated Date) use locale string compare; numeric columns use numeric compare (null treated as −∞).

### Filtering
- A filter input row sits below the header row (sticky).
- Each column has a text input; filtering is case-insensitive substring match on the **displayed** cell value.
- Active filters highlight in amber.
- Filtering and sorting are applied together on every input change.

### Row Count
- A row count bar below the header shows `N of M TIPS` (filtered vs. total).

---

## Export

Button label: **Export CSV**

- Exports the currently **filtered** rows in the current sort order.
- Columns match the display columns exactly (same labels, same formatted values).
- Filename: `tips-reference-<YYYY-MM-DD>.csv` where the date is the settlement date.
- Values containing commas are quoted.

---

## Design

- Stylesheet: `../shared/src/portal-theme.css`
- Full-viewport layout: header (fixed height) + row-count bar + scrollable table fills remaining height.
- No sidebar — all 9 columns are always visible.
- Table uses `border-collapse: separate` with sticky dual-row thead (header + filter).
- Hover row highlight: `#fffde7`.
- No sub-rows, no drilldowns, no modals.

---

## Architecture

Single `index.html` with inline `<script>` — no build step, pure ES5 syntax to match the rest of the portal's static-file pattern. Served directly from GitHub Pages.

---

## Not in Scope

- Editing, rebalancing, or ladder calculations
- Ref CPI date overrides
- Column visibility toggling or reordering
- Charts or sparklines
